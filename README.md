# Foreign Trade Outreach Skill 🌍

B2B外贸开发信全流程系统 - AI驱动的海外客户开发工具。

## 功能
- 🔍 目标客户搜索（6大渠道）
- 👤 买家画像匹配（4种类型）
- ✉️ 个性化开发信生成
- 📧 5封14天跟进序列
- 📊 A/B测试方案
- 📋 客户调研模板
- 📊 飞书多维表格CRM集成

## 支持行业
- 液压设备 (Hydraulic Equipment)
- 阀门 (Valves)
- CNC机械加工 (CNC Machining)
- 更多行业可扩展...

## 使用
将 `foreign-trade-outreach` 目录作为 OpenClaw Skill 使用，或安装 `.skill` 文件。

## 目录结构
```
skills/foreign-trade-outreach/   # Skill本体
cold-email-system/               # 开发信系统文件
  ├── templates/                 # 模板库
  └── prospects/                 # 客户数据
foreign-trade-outreach.skill     # 打包好的Skill文件
```
