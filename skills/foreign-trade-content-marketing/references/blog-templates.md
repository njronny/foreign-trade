# 博客文章模板库

## 模板1: 技术指南型（Ultimate Guide）

**SEO标题格式**: "Ultimate Guide to [Topic] (2026 Update)"
**字数**: 1500-2500字
**结构**:
```
## Introduction（为什么这个话题重要）
## What is [Topic]?（基础定义）
## Key Factors to Consider
### Factor 1
### Factor 2
### Factor 3
## [Topic] Standards & Specifications（行业标准）
## Common Mistakes to Avoid（避坑指南）
## How to Choose the Right [Topic]（选择指南）
## Why [Company] for [Topic]（软性推广）
## FAQ（3-5个常见问题）
## Conclusion + CTA
```

## 模板2: 对比分析型（A vs B）

**SEO标题格式**: "[A] vs [B]: Which is Better for [Application]?"
**字数**: 1000-1500字
**结构**:
```
## Introduction（场景引入）
## What is [A]?（简要说明）
## What is [B]?（简要说明）
## [A] vs [B] Comparison
| Feature | [A] | [B] |
（对比表格5-8行）
## When to Choose [A]
## When to Choose [B]
## Cost Comparison
## Our Recommendation
## Get a Quote（CTA）
```

## 模板3: 列表型（Listicle）

**SEO标题格式**: "[Number] [Topic] Tips/Things/Factors for [Audience]"
**字数**: 800-1500字
**结构**:
```
## Introduction（为什么这些很重要）
## 1. [Tip/Thing]（详细说明+示例）
## 2. [Tip/Thing]
...
## [Number]. [Tip/Thing]
## Summary
## Need Help?（CTA）
```

## 模板4: 客户案例型（Case Study）

**SEO标题格式**: "How We [Result] for [Client Type] in [Industry]"
**字数**: 800-1200字
**结构**:
```
## The Client（匿名背景介绍）
## The Challenge（面临的问题）
### Requirements
- 需求1
- 需求2
## Our Solution（解决方案）
### Process
### Materials
### Quality Control
## The Results（量化成果）
| Metric | Before | After |
## Client Testimonial（客户原话）
## Want Similar Results?（CTA）
```

## 模板5: FAQ型

**SEO标题格式**: "[Topic] FAQ: [Number] Questions Answered"
**字数**: 1000-2000字
**结构**:
```
## Introduction
## 1. [Question]?
[Answer - 3-5句]
## 2. [Question]?
[Answer]
...
## Still Have Questions?（CTA）
```

## SEO写作规则

1. **首段**: 100字内必须出现主关键词
2. **段落**: 每段不超过3-4句
3. **小标题**: H2/H3必须含相关关键词
4. **列表**: 尽量用列表代替长段落（可读性+Featured Snippet机会）
5. **内链**: 每篇至少2-3个内链到产品页/其他博客
6. **外链**: 1-2个权威来源（Wikipedia/行业协会/标准组织）
7. **CTA**: 每篇结尾必须有明确行动号召
8. **图片建议**: 每个H2段落配1张图，标注alt文字建议
