# SEO关键词策略指南

## 关键词研究方法

### 1. 核心关键词发现
搜索公式（使用 exa.web_search_exa）：
```
"[产品] [工艺/服务]" buying guide / cost / supplier
"[产品] manufacturing" process / tolerances / materials
"how to [动作] [产品]" / "best [产品] for [应用]"
"[产品] vs [替代方案]" comparison
```

### 2. 竞品关键词分析
找到同行独立站，分析其页面标题和H1标签：
```
搜索 "site:[竞品域名]" + 关键产品词
```

### 3. 关键词评估矩阵

| 维度 | 评估标准 |
|------|---------|
| **搜索量** | 高=月搜索>1000, 中=100-1000, 低<100 |
| **竞争度** | 高=大品牌占位, 中=混合, 低=机会 |
| **商业意图** | 高=购买意图词, 中=研究词, 低=信息词 |
| **内容机会** | 竞品没有覆盖的=最佳机会 |

### 4. 关键词分类

**按漏斗阶段**：
- **认知阶段（TOFU）**: "what is CNC machining", "types of valves"
- **兴趣阶段（MOFU）**: "CNC machining tolerances guide", "hydraulic valve materials"
- **考虑阶段（BOFU）**: "CNC machining supplier China", "valve body machining cost"
- **决策阶段**: "get CNC quote", "order custom valves online"

**按内容类型**：
- 产品页关键词 → 高商业意图、短尾
- 博客关键词 → 信息意图、长尾
- 案例关键词 → 品牌意图、行业+产品

### 5. 关键词映射到页面

| 页面类型 | 关键词数量 | 示例 |
|---------|-----------|------|
| 首页 | 3-5个核心词 | CNC machining, precision parts |
| 产品分类页 | 5-10个/分类 | CNC milling, CNC turning |
| 单品页 | 3-5个/产品 | aluminum CNC parts, 5-axis machining |
| 博客 | 1-2个/篇 | 长尾词为主 |

### 6. On-Page SEO清单

- [ ] Title tag含关键词（≤60字符）
- [ ] Meta description含关键词（≤160字符）
- [ ] URL简短含关键词
- [ ] H1含主关键词（仅1个）
- [ ] H2/H3含相关关键词
- [ ] 首段100字内出现主关键词
- [ ] 图片alt标签含关键词
- [ ] 内链2-3个（到产品页/其他博客）
- [ ] 外链1-2个（到权威来源）
- [ ] CTA引导询盘
