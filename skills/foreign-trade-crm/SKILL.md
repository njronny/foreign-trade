# 外贸B2B客户跟进&CRM全流程系统

## 前置信息

需要确认（可从上文或直接询问）：
1. **产品/品类** — 具体做什么
2. **目标市场** — 北美/欧洲等
3. **当前客户数量** — 已有多少客户/线索

## 执行流程（按顺序）

### Phase 1: 客户分级体系

**目标**: 科学分类客户，分配不同跟进资源

**参考**: `references/customer-classification.md`

**分级标准**：

| 等级 | 定义 | 特征 | 跟进频率 | 投入资源 |
|------|------|------|---------|---------|
| A级 | 核心客户 | 已下单+返单+高金额 | 每周 | 高（专属PM） |
| B级 | 成长客户 | 已下单/已打样+有潜力 | 每2周 | 中 |
| C级 | 培育客户 | 深度沟通中+有明确需求 | 每月 | 中低 |
| D级 | 长期跟进 | 初步接触/暂无需求 | 每季度 | 低 |
| E级 | 暂停/沉睡 | 超过3个月未回复 | 每6个月 | 最低 |

**打分模型**（总分100）：
| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 采购金额/潜力 | 30% | >$50K=30, $10-50K=20, <$10K=10 |
| 沟通活跃度 | 20% | 本周回复=20, 本月=15, >1月=5 |
| 需求明确度 | 20% | 有图纸+规格=20, 有大致需求=10, 模糊=5 |
| 公司规模 | 15% | 200+人=15, 50-200=10, <50=5 |
| 行业匹配度 | 15% | 完全匹配=15, 部分匹配=10, 不太匹配=5 |

### Phase 2: 跟进节奏管理

**目标**: 每个客户在正确的时间用正确的方式跟进

**参考**: `references/follow-up-rhythm.md`

**跟进节奏表**：

| 客户阶段 | 跟进动作 | 间隔 | 工具 |
|---------|---------|------|------|
| 开发信已发 | 等待回复 | Day 3/7/11/14 | 邮件 |
| 已回复(询价中) | 报价+技术沟通 | 1-2天 | 邮件+电话 |
| 已报价 | 跟进报价反馈 | 3-5天 | 邮件 |
| 谈判中 | 推进决策 | 2-3天 | 邮件+电话 |
| 已下样品单 | 跟踪样品进度 | 每周 | 邮件 |
| 样品已寄出 | 确认收到+反馈 | Day 3/7/14 | 邮件 |
| 样品确认 | 推动量产订单 | 1-3天 | 邮件+电话 |
| 已下量产单 | 跟踪生产+发货 | 每周 | 邮件 |
| 已交付 | 确认满意度 | Day 7/14 | 邮件 |
| 复购客户 | 定期沟通+新品推荐 | 每月 | 邮件+LinkedIn |

### Phase 3: 客户沟通记录模板

**目标**: 标准化记录每次沟通内容

**参考**: `references/communication-log.md`

**沟通记录格式**：
```
日期: YYYY-MM-DD
客户: [公司名] — [联系人]
方向: 主动/被动
方式: 邮件/电话/LinkedIn/WhatsApp
内容摘要: [3-5句概括]
关键信息: [客户提到的重要信息]
下一步: [具体行动+截止日期]
状态变化: [如: C级→B级]
```

### Phase 4: 复购预测模型

**目标**: 预测哪些客户即将需要复购

**参考**: `references/reorder-prediction.md`

**复购信号**：
1. 上次订单消耗完毕（根据数量+行业消耗率推算）
2. 客户问"交期能不能更快"（暗示库存下降）
3. 客户要求修改图纸或增加新规格（新产品上线）
4. 季节性因素（如Q4旺季前备货）
5. 行业展会前（样品需求增加）

**复购提醒规则**：
| 客户类型 | 预计消耗周期 | 提醒时间 |
|---------|------------|---------|
| 月消耗型 | 30天 | Day 20 |
| 季度消耗型 | 90天 | Day 70 |
| 半年消耗型 | 180天 | Day 150 |
| 年度消耗型 | 365天 | Day 300 |

### Phase 5: 流失预警&挽回

**目标**: 识别即将流失的客户并采取行动

**参考**: `references/churn-prevention.md`

**流失预警信号**：
1. 🚨 超过30天未回复邮件（之前回复很及时）
2. 🚨 报价后超过2周无反馈
3. 🚨 突然降低订单量或频率
4. 🚨 取消已确认的订单
5. 🚨 社交媒体不再互动

**挽回策略**：
| 场景 | 策略 | 话术 |
|------|------|------|
| 不回复邮件 | 换渠道（LinkedIn/WhatsApp） | "Hi, just checking in via a different channel..." |
| 价格原因流失 | 重新报价+优化方案 | "We've found a way to reduce cost by [X]%..." |
| 供应商切换 | 了解原因+留门 | "No worries. We're here if you ever need a backup." |
| 项目暂停 | 保持低频联系 | "Just sharing an industry update..." |

## 关键原则

1. **数据即资产** — 客户信息是公司最重要的资产，必须记录
2. **不依赖个人** — 任何业务员离职，客户信息不丢失
3. **及时记录** — 每次沟通后24小时内更新记录
4. **定期回顾** — 每周回顾客户状态，调整跟进策略
5. **分层管理** — A级客户投入最多资源，E级最低
6. **长期主义** — 外贸客户成交周期1-6个月，坚持跟进

## 与其他3个Skill联动

```
Skill 1 (开发信) → 获取线索 → 导入CRM
Skill 2 (内容营销) → 触达客户 → 记录互动
Skill 3 (询盘报价) → 报价谈判 → 记录状态
Skill 4 (客户跟进) → 持续管理 → 复购+防流失
```
