# 沟通记录模板

## 标准记录格式

```
📅 日期: 2026-04-19
🏢 客户: [公司名] — [联系人]
📧 方式: 邮件/电话/LinkedIn/WhatsApp
📥 方向: 主动跟进 / 客户主动联系

📝 内容摘要:
[3-5句概括沟通内容，包含关键数据和结论]

🔑 关键信息:
- [客户提到的重要信息1]
- [客户提到的重要信息2]
- [隐含信息/推测]

📋 下一步:
- [ ] 动作1 (截止: YYYY-MM-DD)
- [ ] 动作2 (截止: YYYY-MM-DD)

📊 状态变化:
客户等级: [原等级] → [新等级]
客户状态: [原状态] → [新状态]
```

## 记录示例

### 示例1: 收到询盘
```
📅 2026-04-19
🏢 RED-WHITE VALVE — Jeremy Beck
📧 邮件
📥 客户回复开发信

📝 Jeremy回复了我们的开发信，要求报VALVE-285阀体。
有STEP图纸，要求50pcs试单+200-500/月量产。
材料球墨铸铁，客户提供毛坯。

🔑 关键信息:
- 他之前看过我们的邮件，说明开发信有效
- 提供铸件毛坯 = 他们有自己的铸造供应商
- 月产200-500 = 潜力$10-30K/月
- 要求CMM报告 = 质量意识高

📋 下一步:
- [x] 技术评估 (截止: 04-20)
- [ ] 发送报价 (截止: 04-21)

📊 变化:
等级: D → B (从"长期跟进"升级到"成长客户")
状态: 长期跟进 → 报价中
```

### 示例2: 报价后跟进
```
📅 2026-04-25
🏢 RED-WHITE VALVE — Jeremy Beck
📧 邮件
📥 主动跟进

📝 发送报价后第3天，主动跟进。Jeremy回复说价格偏高，
正在和意大利母公司供应商对比。我提出可以做成本拆解对比。

🔑 关键信息:
- 他没有直接拒绝 = 有兴趣
- 在和意大利比价 = 价格是关键决策因素
- 没有说交期问题 = 可能交期不是痛点

📋 下一步:
- [ ] 准备成本对比分析 (截止: 04-26)
- [ ] 提出优化方案降低$5-10/pc (截止: 04-27)

📊 变化:
状态: 报价中 → 谈判中
```

## 记录规则

1. **24小时内记录** — 超过24小时记忆会模糊
2. **客观记录** — 记录事实，区分"客户说的"和"我推测的"
3. **关键信息用引号** — 客户原话比概括更有价值
4. **标注下一步** — 每条记录必须有明确的下一步
5. **关联文件** — 如有报价单/图纸，标注文件位置
