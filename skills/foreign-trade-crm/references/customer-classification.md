# 客户分级体系

## 评分模型（总分100分）

### 维度1: 采购金额/潜力（30分）
| 指标 | 分数 |
|------|------|
| 年采购额 > $100K 或明确大项目 | 30 |
| 年采购额 $50-100K | 25 |
| 年采购额 $10-50K | 20 |
| 年采购额 < $10K | 10 |
| 仅询价未下单 | 5 |

### 维度2: 沟通活跃度（20分）
| 指标 | 分数 |
|------|------|
| 本周内回复过 | 20 |
| 本月内回复过 | 15 |
| 1-3个月内回复过 | 10 |
| 3-6个月未回复 | 5 |
| >6个月未回复 | 0 |

### 维度3: 需求明确度（20分）
| 指标 | 分数 |
|------|------|
| 有图纸+规格+数量+交期 | 20 |
| 有图纸+大致规格 | 15 |
| 有需求描述无图纸 | 10 |
| 仅泛泛询问 | 5 |

### 维度4: 公司规模（15分）
| 指标 | 分数 |
|------|------|
| 200+人，年营收$50M+ | 15 |
| 50-200人，年营收$10-50M | 10 |
| 11-50人，年营收$1-10M | 7 |
| <10人 | 5 |

### 维度5: 行业匹配度（15分）
| 指标 | 分数 |
|------|------|
| 完全匹配我们的核心能力 | 15 |
| 大部分匹配 | 10 |
| 部分匹配，需要扩展能力 | 5 |
| 不太匹配 | 0 |

## 分级标准

| 等级 | 分数范围 | 客户数占比 | 资源分配 |
|------|---------|-----------|---------|
| A级 核心客户 | 80-100 | 5-10% | 专属PM，优先排产 |
| B级 成长客户 | 60-79 | 15-20% | 定期主动跟进 |
| C级 培育客户 | 40-59 | 30-40% | 按节奏跟进 |
| D级 长期跟进 | 20-39 | 20-30% | 季度触达 |
| E级 暂停/沉睡 | <20 | 10-20% | 6个月一次 |

## 等级升降规则

### 升级条件
- C→B: 首次下单或确认样品
- B→A: 连续3次返单或单笔订单>$20K
- D→C: 提供具体需求/图纸

### 降级条件
- A→B: 3个月未下单
- B→C: 2个月未回复
- C→D: 3个月未回复
- D→E: 6个月未回复

## 每个等级的跟进SOP

### A级客户（每周）
- 周一: 回顾上周沟通，规划本周
- 周三: 主动联系（邮件/电话）
- 周五: 分享行业资讯或新品信息
- 每月: 一次视频会议/电话
- 每季度: 一次工厂参观邀请

### B级客户（每2周）
- 主动跟进最新需求
- 分享相关案例
- 推荐新品/新能力

### C级客户（每月）
- 邮件跟进（结合内容营销）
- LinkedIn互动
- 节日问候

### D级客户（每季度）
- 季度行业报告分享
- 新产品/新能力通知
- 展会邀约

### E级客户（每6个月）
- "我们还在"问候邮件
- 重大更新通知
- 不投入太多资源
