# 外贸B2B询盘处理&报价全流程系统

## 前置信息

需要确认（可从上文或直接询问）：
1. **产品/品类** — 具体做什么
2. **目标市场** — 北美/欧洲等
3. **公司信息** — 名称、邮箱、官网

## 执行流程（按顺序）

### Phase 1: 询盘解析

**目标**: 快速理解客户需求，判断询盘质量

**步骤**：
1. 提取关键信息：产品类型、材料、数量、公差、表面处理、交期要求
2. 判断询盘质量（A/B/C类分级）
3. 识别客户类型（采购/工程师/老板）

**询盘质量分级**：
| 等级 | 特征 | 响应策略 |
|------|------|---------|
| A类 | 有图纸+明确规格+数量+交期 | 优先回复，24小时内出完整报价 |
| B类 | 有大致需求但缺细节 | 回复+追问确认关键参数 |
| C类 | 泛泛询问"你能做X吗" | 简短回复确认能力+引导提供图纸 |

**参考**: `references/inquiry-analysis.md`

**产出**: 询盘分析报告

### Phase 2: 技术评估

**目标**: 评估工艺可行性、识别风险点

**步骤**：
1. 审查图纸（如果有STEP/IGES/DWG）
2. 确定加工工艺路线
3. 评估材料可获得性和成本
4. 识别DFM问题（难加工特征/公差过严/不合理设计）
5. 确定检测方案

**技术评估清单**：
- [ ] 材料是否可获得？交货周期？
- [ ] 公差要求是否在能力范围内？
- [ ] 是否需要特殊刀具或夹具？
- [ ] 表面处理是否需要外协？
- [ ] 有无DFM优化建议？
- [ ] 检测方案：CMM/通止规/目视？

**参考**: `references/technical-assessment.md`

**产出**: 技术评估报告（含DFM建议）

### Phase 3: 成本核算

**目标**: 精准计算单件成本

**成本结构**：
```
总成本 = 材料费 + 加工费 + 夹具/ setup费 + 后处理费 + 检测费 + 包装运费 + 利润
```

**参考**: `references/cost-calculation.md`

**成本计算方法**：

| 成本项 | 计算公式 | 备注 |
|--------|---------|------|
| 材料费 | 单件毛坯重量 × 材料单价 × (1+损耗率) | 损耗率1.5-5x |
| 加工费 | 单件工时 × 机床小时费率 | 3轴$25-40/hr, 5轴$40-60/hr |
| 夹具费 | 夹具总成本 ÷ 订单数量 | 首批高，后续摊薄 |
| 后处理 | 按工艺单价 | 阳极氧化$3-8/件 |
| 检测费 | CMM报告$5-20/件 | 可选全检或抽检 |
| 包装运费 | 按重量和方式 | 空运快/海运便宜 |
| 利润 | 总成本 × 利润率(15-30%) | 根据竞争情况调整 |

### Phase 4: 报价单生成

**目标**: 生成专业的英文报价单

**参考**: `references/quotation-template.md`

**报价单结构**：
1. 公司抬头（Logo/名称/联系方式）
2. 报价单号+日期+有效期
3. 客户信息
4. 产品明细（图号/名称/材料/数量/单价/总价）
5. 交货条款（FOB/CIF/DDP）
6. 付款条款（T/T 30%+70% / L/C 等）
7. 交货周期
8. 备注（公差/表面处理/包装等）
9. DFM建议（如有）
10. CTA

**报价策略**：
| 场景 | 策略 |
|------|------|
| 首次报价 | 报中等偏上价格，留谈判空间 |
| 竞争性报价 | 精准成本+薄利，突出质量和服务 |
| 样品报价 | 样品费可收可免，但运费必收 |
| 长期客户 | 阶梯报价（100/500/1000+） |

### Phase 5: 多轮谈判话术

**目标**: 应对客户常见压价和谈判场景

**参考**: `references/negotiation-scripts.md`

**常见场景及应对**：
1. "Your price is too high" → 不降价，改为拆解成本/提供降本方案
2. "Can you match XX price?" → 追问对方报价细节，找差异
3. "We need a lower price for first order" → 样品价优惠+量产恢复原价
4. "What's your MOQ?" → 灵活MOQ+阶梯报价
5. "Can you do DDP?" → 可以，但计入物流成本
6. "We need samples first" → 免费样品+运费到付

### Phase 6: 样品管理

**目标**: 规范样品流程，提高转化率

**参考**: `references/sample-management.md`

**样品流程**：
1. 确认样品要求（材料/公差/表面处理/数量）
2. 报样品费+运费（或免样品费收运费）
3. 制作样品（7-15天）
4. 全尺寸检测+拍照+检测报告
5. 寄出+追踪快递
6. 跟进客户确认

**样品策略**：
| 策略 | 适用场景 | 风险 |
|------|---------|------|
| 免费样品+运费到付 | 高质量询盘 | 运费风险 |
| 收样品费+运费 | 普通询盘 | 低 |
| 免费样品+免运费 | 大客户/战略客户 | 成本风险 |
| 收样品费，下单后退 | 意向明确的客户 | 现金流 |

## 关键原则

1. **24小时响应** — 询盘回复速度直接影响成交率
2. **报价要专业** — 分项报价 > 一口价，展示透明度
3. **不急于降价** — 先解释价值，再讨论价格
4. **样品是关键** — 免费样品是CNC行业最有效的转化手段
5. **记录一切** — 每次沟通都记录在案
6. **DFM增值** — 在报价中提供设计优化建议，展示专业度
