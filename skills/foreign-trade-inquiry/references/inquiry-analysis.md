# 询盘解析指南

## 信息提取模板

收到询盘后，立即提取以下关键信息：

| 信息项 | 说明 | 如果缺失 |
|--------|------|---------|
| **产品名称/图号** | 客户要做什么 | 追问 |
| **材料** | 具体牌号（如AL 6061-T6） | 建议常用材料 |
| **数量** | 样品数+量产数 | 追问，无法报价 |
| **图纸** | STEP/IGES/DWG/PDF | 追问 |
| **公差要求** | 通用还是特殊 | 默认ISO 2768-mK |
| **表面处理** | 阳极氧化/镀锌/喷漆等 | 默认as-machined |
| **交期要求** | 期望交付日期 | 给标准交期 |
| **目标价格** | 客户预算（如有） | 按标准报价 |
| **目的地** | 用于计算运费 | 追问 |
| **付款方式** | T/T, L/C, PayPal等 | 默认T/T 30%+70% |

## 询盘来源与质量

| 来源 | 典型质量 | 响应策略 |
|------|---------|---------|
| 开发信回复 | A/B类，高意向 | 24小时内详细回复 |
| 中国制造网RFQ | B/C类，比价为主 | 快速报价+突出优势 |
| 展会名片 | B类，有面谈基础 | 提到展会见面+报价 |
| 独立站表单 | B类，主动找上门 | 重点回复+电话跟进 |
| 客户转介绍 | A类，信任度高 | 优先处理 |

## 快速回复模板

### 有图纸的询盘（A类）
```

Before I provide a detailed quote, a few quick questions:
1. [确认材料]
2. [确认数量 — prototype + production?]
3. [确认表面处理]
4. [确认交期要求]

I'll send you a complete quotation within 24 hours once confirmed.

Meanwhile, here's a brief overview of our capabilities for this type of part:
- [列出1-2个相关能力]

Best regards,
[Name]
```

### 无图纸的询盘（B/C类）
```
Hi [Name],

Thanks for reaching out! We do manufacture [产品类型] and would love to help.

To provide an accurate quote, could you share:
1. A drawing or sketch (STEP, IGES, DWG, or even a PDF)
2. Material preference
3. Required quantity
4. Any specific tolerances or surface finish requirements

If you don't have a drawing yet, I can also provide a general price range based on similar parts we've made.

Best regards,
[Name]
```
