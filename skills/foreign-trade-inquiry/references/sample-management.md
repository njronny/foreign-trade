# 样品管理流程

## 样品流程 SOP

### Step 1: 确认需求
```
需确认:
- [ ] 图纸版本（最终版？）
- [ ] 材料牌号
- [ ] 公差要求
- [ ] 表面处理
- [ ] 样品数量（通常1-3件）
- [ ] 是否需要特殊包装
```

### Step 2: 报价
| 项目 | 费用 | 说明 |
|------|------|------|
| 样品费 | $XX-XXX/件 | 根据复杂度 |
| 运费 | $50-100 | DHL/FedEx到北美 |
| 总计 | $XXX | 样品费+运费 |

### Step 3: 制作（7-15天）
- 编程 → 首件加工 → 全尺寸检测 → 照片记录 → 包装

### Step 4: 发货前确认
发以下资料给客户确认后再寄出：
- [ ] 成品照片（6面+细节）
- [ ] CMM检测报告
- [ ] 包装照片
- [ ] 快递单号

### Step 5: 跟进
```
寄出后:
Day 1: "Samples shipped! Tracking #: XXXXX. Expected delivery: [date]"
Day 3-5: (预计到达日) "Hope the samples arrived safely. Let me know if you need any clarification on the inspection report."
Day 7: "Have you had a chance to evaluate the samples? Any feedback?"
Day 14: (如无回复) "Just following up on the samples. Happy to discuss any adjustments or move forward with production planning."
```

## 样品到订单转化技巧

1. **附带DFM建议** — 在样品报告里加优化建议，展示专业度
2. **批量报价一起发** — 样品寄出时同时附上100/500/1000+的阶梯报价
3. **提供材料证书** — 增加信任感
4. **设置有效期** — "Production pricing valid for 60 days from sample shipment"
5. **主动问反馈** — 不要等客户来找你
