# 技术评估清单

## 图纸审查要点

### 1. 尺寸与公差
- [ ] 是否有超出能力的公差？（±0.005mm以下需特别注意）
- [ ] 公差标注是否完整？未标注公差按ISO 2768-mK
- [ ] 是否有过度的GD&T要求？

### 2. 材料评估
- [ ] 材料牌号是否明确？
- [ ] 材料是否可采购？交货周期？
- [ ] 是否有替代材料建议？

| 材料 | 机加工性 | 刀具磨损 | 热处理 |
|------|---------|---------|--------|
| AL 6061 | 极好 | 低 | 可热处理 |
| AL 7075 | 好 | 中 | 已热处理 |
| SS 304 | 中等 | 中高 | 不可 |
| SS 316 | 中等 | 中高 | 不可 |
| SS 17-4PH | 中等 | 高 | 沉淀硬化 |
| 钛合金 | 困难 | 高 | 可 |
| 黄铜 | 极好 | 低 | 不可 |
| PEEK | 中等 | 低 | 不可 |

### 3. 工艺路线规划
```
毛坯 → 粗加工 → 热处理(可选) → 精加工 → 表面处理 → 检测 → 包装
```

需确定：
- [ ] 几次装夹？（每次装夹增加成本+累积误差）
- [ ] 是否需要特殊夹具？
- [ ] 是否需要外协（热处理/表面处理/电镀）？

### 4. DFM优化建议

**常见可优化点**：

| 问题 | DFM建议 | 节省 |
|------|---------|------|
| 深孔(L/D>8:1) | 改为通孔或增加钻孔工艺 | 降低报废率 |
| 薄壁(<1mm) | 增加壁厚或改用铸件 | 降低难度 |
| 锐角内腔 | 增加圆角R0.5mm | 减少刀具磨损 |
| 过严公差(±0.002mm) | 仅密封面精密，其余放宽 | 降低成本30%+ |
| 多方向孔 | 改为可一次装夹加工的设计 | 减少装夹次数 |

### 5. 检测方案

| 特征 | 检测方法 | 工具 |
|------|---------|------|
| 线性尺寸 | 抽检/全检 | 卡尺、千分尺 |
| 孔径 | 全检 | 内径千分尺/通止规 |
| 位置度 | 全检 | CMM |
| 表面粗糙度 | 抽检 | 粗糙度仪 |
| 硬度 | 抽检 | 硬度计 |
| 外观 | 全检 | 目视 |
