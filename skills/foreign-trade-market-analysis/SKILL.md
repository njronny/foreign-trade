# 外贸B2B市场分析&竞品分析全流程系统

## 前置信息

需要确认（可从上文或直接询问）：
1. **产品/品类** — 具体做什么（如CNC机械加工）
2. **目标市场** — 北美/欧洲等
3. **公司信息** — 名称、优势、规模
4. **现有客户** — 已有哪些行业/客户（如有）

## 执行流程（按顺序）

### Phase 1: 市场规模&趋势

**目标**: 量化目标市场，判断值不值得做

**步骤**：
1. 使用 web_fetch 搜索市场规模数据
2. 分析增长趋势和驱动因素
3. 识别细分市场机会

**搜索关键词**：
- "CNC machining market size North America 2025 2026"
- "precision machining outsourcing market size"
- "contract manufacturing market size USA"
- "[行业] components outsourcing market size"

**市场报告结构**：
| 指标 | 数据 |
|------|------|
| 全球市场规模 | $XXB (2025) |
| 北美市场规模 | $XXB |
| 年增长率(CAGR) | X.X% |
| 主要驱动因素 | 1. 2. 3. |
| 细分市场Top3 | 行业1/行业2/行业3 |
| 外包比例 | XX%企业选择外包 |

**参考**: `references/market-sizing.md`

**产出**: 市场规模报告

### Phase 2: 目标行业筛选

**目标**: 从所有可能的行业中，选出最值得切入的2-3个

**步骤**：
1. 列出所有相关行业
2. 用评分模型打分
3. 选出Top 2-3个优先行业

**行业评分模型**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 市场规模 | 25% | 这个行业CNC外包有多大 |
| 增长速度 | 20% | 是否在增长 |
| 竞争强度 | 20% | 中国供应商多不多 |
| 我们的能力匹配度 | 20% | 是否在能力范围内 |
| 利润空间 | 15% | 行业平均毛利 |

**行业筛选输出**：

| 行业 | 市场规模 | 增长 | 竞争 | 匹配 | 利润 | 总分 | 排名 |
|------|---------|------|------|------|------|------|------|
| 液压设备 | — | — | — | — | — | — | 1 |
| 阀门制造 | — | — | — | — | — | — | 2 |
| ... | — | — | — | — | — | — | — |

**参考**: `references/industry-screening.md`

**产出**: 行业优先级排名

### Phase 3: 竞品地图

4. **低端竞品** — 印度/东南亚供应商

### Phase 4: 价格带分析

**目标**: 搞清楚市场价格体系，找到我们的定价空间

**价格带模型**：

```
低价 ←————————————————————————→ 高价
 |                                    |
印度/东南亚    中国      台湾     北美本地
$15-25/hr    $25-50   $40-70   $75-150/hr
 |                                    |
低质量        性价比    品质平衡   快速响应
```

**需要搞清楚**：
1. 各层级典型报价（按工时/按件）
2. 我们在哪一层？上下各是谁？
3. 客户从中国供应商切换到我们的理由是什么？
4. 客户从北美本地切换到我们的理由是什么？

**参考**: `references/pricing-analysis.md`

**产出**: 价格矩阵 + 定价建议

### Phase 5: 差异化定位

**目标**: 明确我们是谁、服务谁、凭什么赢

**定位画布**：
```
┌──────────────────────────────────────┐
│           WHO（服务谁）               │
│  北美中小型设备制造商的精密零件需求    │
├──────────────────────────────────────┤
│           WHAT（做什么）              │
│  CNC精密加工，专注液压/阀门行业零件    │
├──────────────────────────────────────┤
│           WHY（凭什么）               │
│  行业Know-how + 英文沟通 + 质量体系    │
├──────────────────────────────────────┤
│           HOW（怎么做）               │
│  DFM增值 + 透明报价 + 快速响应        │
└──────────────────────────────────────┘
```

**USP提炼方法**：
1. 列出我们所有优势
2. 对比竞品，找出"只有我们有"的
3. 筛选出客户最关心的2-3个
4. 包装成一句话USP

**参考**: `references/positioning.md`

**产出**: 定位文档 + USP + Elevator Pitch

## 与其他5个Skill的关系

```
Skill 0 (市场分析) → 指导所有后续Skill
    ├→ Skill 1 (开发信): 卖点/目标客户/差异化话术
    ├→ Skill 2 (内容营销): 关键词/话题/渠道
    ├→ Skill 3 (询盘报价): 定价策略/谈判筹码
    └→ Skill 4 (CRM): 客户分级权重/跟进策略
```

**建议执行顺序**: 先做市场分析（本Skill），再做其他4个。

## 关键原则

1. **数据驱动** — 用真实数据说话，不靠猜
2. **聚焦** — 选2-3个行业做深，不要撒网
3. **差异化** — 不做"又一家中国机加工厂"
4. **可执行** — 分析的目的是行动，不是PPT
5. **定期更新** — 市场在变，每季度回顾一次
