# 渠道分析

## 北美B2B采购行为

### 买家怎么找供应商？
| 渠道 | 使用率 | 转化率 | 成本 | 说明 |
|------|--------|--------|------|------|
| Google搜索 | 72% | 3-5% | 中 | 最大流量入口 |
| 行业展会 | 45% | 10-20% | 高 | 最信任 |
| 同行推荐 | 38% | 30-50% | 无 | 最高转化 |
| LinkedIn | 25% | 2-5% | 低 | 增长快 |
| B2B平台(Thomas Net等) | 22% | 5-10% | 中 | 专业买家 |
| 中国制造网 | 18% | 1-3% | 中 | 比价为主 |
| Facebook/Instagram | 8% | <1% | 低 | 不适合 |
| 冷邮件 | — | 1-5% | 低 | 主动出击 |

### 采购决策流程
```
1. 需求产生 → 采购/工程师搜索
2. 初筛 → 5-10个候选（Google/推荐/展会）
3. 深入了解 → 看网站/案例/认证
4. RFQ → 发给3-5家
5. 样品评估 → 选1-2家打样
6. 小批量试产 → 验证稳定性
7. 量产 → 锁定1-2家长期合作
```
