## 竞品分析维度

### 1. 网站分析
- 加载速度
- 英文质量
- 信任元素（认证/案例/评价）
- CTA设计
- 在线报价功能
- 产品页完整度

### 2. SEO分析
```
使用工具（免费）:
- Google搜索目标关键词 → 看谁排名前10
- Similar Web（免费版）→ 看流量估算
- 搜索 "site:竞品.com" → 看收录页面数
```

### 3. 内容分析
- 有没有博客？
- 内容质量如何？
- 更新频率？
- 社交媒体活跃度？

### 4. 定价分析
- 公开价格？还是"Request a Quote"？
- 有没有在线报价器？
- MOQ？
- 交期承诺？

### 5. 客户评价
```
搜索:
- "[竞品名] review"
- "[竞品名] Reddit"
- "[竞品名] complaint"
- Google Maps评论（如有）
- G2/Clutch评价
```

## 从竞品分析到差异化

### 找差距
1. 竞品普遍做得差的 → 我们的机会
2. 客户抱怨最多的 → 我们的切入点
3. 没人做的 → 差异化方向

### 常见差异化方向
| 方向 | 适合 | 不适合 |
|------|------|--------|
| 行业专精（只做液压） | 有行业经验 | 想做通才 |
| 超快响应（2小时报价） | 流程高效 | 产能紧张 |
| DFM增值服务 | 有工程能力 | 纯代工 |
| 小批量/样品友好 | 灵活排产 | 起定量大 |
| 100%英文沟通 | 团队英语好 | 沟通差 |
