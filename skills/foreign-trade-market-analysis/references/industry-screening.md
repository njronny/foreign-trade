## 评分标准

### 市场规模（25分）
| 分数 | 标准 |
|------|------|
| 25 | 北美外包市场 > $5B |
| 20 | $1-5B |
| 15 | $500M-1B |
| 10 | $100-500M |
| 5 | < $100M |

### 增长速度（20分）
| 分数 | 标准 |
|------|------|
| 20 | CAGR > 8% |
| 15 | CAGR 5-8% |
| 10 | CAGR 3-5% |
| 5 | CAGR < 3% |
| 0 | 负增长 |

### 竞争强度（20分）
| 分数 | 标准 |
|------|------|
| 20 | 中国供应商少，蓝海 |
| 15 | 有一些，但不拥挤 |
| 10 | 较多，红海 |
| 5 | 拥挤，价格战 |

### 能力匹配度（20分）
| 分数 | 标准 |
|------|------|
| 20 | 完全匹配现有设备和经验 |
| 15 | 大部分匹配，需少量扩展 |
| 10 | 部分匹配，需投资 |
| 5 | 不太匹配 |

### 利润空间（15分）
| 分数 | 标准 |
|------|------|
| 15 | 毛利 > 35% |
| 10 | 毛利 25-35% |
| 5 | 毛利 < 25% |
