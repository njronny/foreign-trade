# 市场规模研究方法

## 数据来源

### 免费数据源
1. **Google搜索** — "CNC machining market size report"
2. **Statista** — 部分免费数据
3. **IBISWorld** — 摘要免费
4. **Grand View Research** — 报告摘要
5. **行业展会数据** — 参展商/观众统计
6. **海关数据** — 出口统计（部分免费）

### 搜索关键词模板
```
"[品类] market size [地区] [年份]"
"[品类] outsourcing market size"
"[品类] contract manufacturing market"
"[品类] industry report [地区]"
"[品类] TAM SAM SOM"
```

## 市场规模估算方法

### 方法1: 自上而下（Top-Down）
```
全球CNC加工市场 → 北美占比 → 外包占比 → 我们可触达的部分
$100B × 35% × 25% × 0.1% = $8.75M (TAM → SAM → SOM)
```

### 方法2: 自下而上（Bottom-Up）
```
目标客户数 × 平均年采购额
= 5,000家潜在客户 × $20K/年 = $100M (SAM)
```

### 方法3: 竞品收入推算
```
头部竞品年收入 × 市场份额估算 = 市场规模
Xometry: ~$500M revenue → 占即时报价平台~30% → 平台市场~$1.7B
```

## TAM/SAM/SOM框架

| 层级 | 定义 | CNC机械加工示例 |
|------|------|---------------|
| TAM | 全球总需求 | 全球CNC外包 $100B+ |
| SAM | 可服务市场 | 北美CNC外包 $25-35B |
| SOM | 可获得市场 | 北美液压/阀门CNC外包 $500M-1B |

## 关键指标

分析市场时要搞清楚：
1. **市场规模** — $XXB（现在）
2. **增长率** — CAGR X.X%
3. **驱动因素** — 什么在推动增长
4. **阻碍因素** — 什么在限制增长
5. **外包率** — 多少企业选择外包
6. **价格趋势** — 涨价还是降价
7. **技术趋势** — 5轴/自动化/数字化
8. **供应链变化** — 脱钩/近岸/友岸
