## 我们的价格定位策略

### 定位选择

```
选项A: "性价比之王" — 定价在中国普通和优秀之间
  优势: 容易获客
  劣势: 利润薄，竞争激烈

选项B: "优质中价" — 定价在中国优秀和台湾之间 ← 推荐
  优势: 有利润空间，差异化明显
  劣势: 需要证明质量

选项C: "高端中国" — 定价接近台湾
  优势: 高利润
  劣势: 获客难，需要强品牌
```

### 定价公式
```
报价 = 成本 × (1 + 利润率)

利润率建议:
- 样品/试单: 25-30%（高setup成本）
- 标准订单: 20-25%
- 长期返单: 15-20%
- 竞争性订单: 10-15%（不亏本）

最低线: 绝不低于成本的110%
```
