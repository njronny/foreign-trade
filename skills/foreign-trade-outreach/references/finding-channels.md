# 6大找客户渠道

## 1. Google高级搜索（免费）
搜索公式：
```
"[产品关键词]" "RFQ" site:thomasnet.com
"[产品关键词]" "purchasing" site:linkedin.com
"[产品关键词]" "sourcing" OR "procurement" "USA"
"looking for" "[产品关键词] supplier"
```

## 2. Thomas Net（免费）
- thomasnet.com — 北美最大工业目录
- 搜索产品关键词 → 筛选Companies
- 找OEM制造商和装配厂

## 3. LinkedIn（免费/$99/月）
搜索条件：
- 职位：Purchasing/Sourcing/Procurement Manager
- 行业：按目标行业筛选
- 地区：United States
- 公司规模：11-50人、51-200人

## 4. 展会参展商名单（免费金矿）
- 找目标行业的展会 → 官网 exhibitor list
- 参展商 = 潜在客户（都需要供应链）
- 北美重点：IMTS、FABTECH、WESTEC、EASTEC、AERODEF

## 5. 海关数据（付费）
- ImportGenius / Panjiva / 52wmb.com
- 找正在从中国进口同类产品的公司
- 这些公司已经在从中国采购 = 最佳目标

## 6. B2B平台RFQ（免费）
- Alibaba.com → "Buying Requests"
- Made-in-China.com → "采购信息"
- GlobalSources.com → 买家求购
