# 客户调研5步模板

## 第一步：公司信息
公司全名、所在州/城市、所属行业、公司规模、年营收、主要产品
来源：Google、LinkedIn

## 第二步：关键决策人
采购经理姓名/职位/在职时间/邮箱
工程负责人姓名/职位
来源：LinkedIn

## 第三步：竞品/供应商线索
现有供应商、参加的展会、社交媒体动态
来源：Google搜索、展会官网

## 第四步：触发事件（回复率3-5倍提升）
- 新产品发布 / 融资扩张
- 投诉现有供应商
- 新员工入职（采购岗）
- 季节性采购周期
来源：官网新闻、LinkedIn、Crunchbase

## 第五步：个性化钩子
```
[Company] 最近 [触发事件]，可能需要 [推测需求]。
联系 [Key Person]，因为 [原因]。
邮件中提到 [具体细节] 展示做了功课。
```

## 推荐工具
- Hunter.io — 查邮箱（免费50次/月）
- RocketReach — 查联系方式（$49/月）
- LinkedIn Sales Navigator — 精准找人（$99/月）
- SimilarWeb — 网站流量（免费基础版）
- Crunchbase — 融资信息（免费基础版）
